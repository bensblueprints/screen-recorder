Features

Everything you need, nothing you don't

Dual-Stream Recording

Screen and webcam are captured as separate MP4 files in the same session folder. Edit them independently or combine them however you want.

screen.mp4 + webcam.mp4

Hardware Encoding

Automatically detects and uses NVENC (NVIDIA) or VideoToolbox (Mac) for GPU-accelerated H.264 encoding. Falls back to software encoding gracefully.

2-5% CPU usage

Live Webcam Preview

See your webcam feed in real-time while recording. Side-by-side preview panels let you monitor both streams at a glance.

Stack Export

One-click export that stacks screen on top and webcam on bottom in 9:16 portrait format. Perfect for TikTok, Reels, and Shorts.

9:16 portrait

Cross-Platform

Runs on Windows and macOS. Bundled FFmpeg handles all the heavy lifting so there's nothing extra to install.

Session Folders

Each recording session gets its own timestamped folder. Screen and webcam files are synced and organized automatically.

rec_2026-03-17_14-30-00/

How It Works

Record in three steps

1

Select Sources

Pick your screen, webcam, and microphone from the dropdowns. The app detects all available devices automatically.

2

Hit Record

Click the red record button. FFmpeg captures both streams simultaneously using hardware encoding for minimal CPU usage.

3

Export & Share

Stop recording to get separate MP4 files. Use Stack Export for an instant 9:16 social media video, or edit the files yourself.
